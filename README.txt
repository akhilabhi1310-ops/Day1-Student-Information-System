# Student Information System

## Description
A Python-based Student Information System that stores student details and calculates academic performance.

## Features
- Student Details Input
- Subject Marks Input
- Total Marks Calculation
- Average Marks Calculation
- Percentage Calculation
- Grade Calculation

## Concepts Used
- Variables
- Input and Output
- Type Casting
- Arithmetic Operators
- Conditional Statements (if-elif-else)
